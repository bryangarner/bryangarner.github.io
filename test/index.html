Florida Keys Reef Guide

Data sources:

Florida Geographic Data Library, Data.gov, Noaa.gov

Development libraries:

Leaflet, Panolens, jQuery, JavaScript, HTML, CSS

Software used:

Brackets, GitHub, ArcGIS Desktop

Completed August 2019


The goal of this project was to create an interactive web map that allowed users to explore well known and established coral reef sites within the Florida Keys National Marine Sanctuary. The interactive map was an effort to modernize the many static map resources that can be found scattered across the internet. One key aspect of the modernization was to incorporate a mechanism to allow users to view each reef site or marine sanctuary zone in a 360-degree environment. In addition, give the users an opportunity to read information about said reef. The intended audience for this web map ranges from first time reef explorers to committed environmentalists. This web map is nested in a larger pseudo organizationâ€™s website called the Florida Keys Reef Guide. This web page provides users with condensed, organized, tips that at a minimum if abided by, will ensure the reefs will be better taken care of for the future. The webpage also includes links to organizations whose primary mission is to care for and educate about coral reefs and coral reefs health.

Users can search for specific features by name or click any feature to get more infomration pertaining to the area. Users also have the ability to view reefs and marine zone areas in a 360-degree environment.

The site was developed with Leaflet for the mapping component along with Panolens for the 360-degree photo viewer. Additional leaflet plugins were also used for the search and the baselayer toggle functions.
